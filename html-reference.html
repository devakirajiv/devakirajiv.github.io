HTML

The start tag for every HTML code is <html>, and the close tag is </html>. Your HTML code will not run without these tags at the beginnind and end of your code.

HTML

Titles

The <title> tag is required in all HTML documents and it defines the title of the document. The text of your title will immediately be followed by the end tag, </title>

Titles

Headings

You can use different headings, <h1> and <h2>, etc to visibly title your paragraphs. These also need end tags.

Headings

Body

The <body> tag defines the document's body. It contains your paragraphs and headings, and then it needs an end tag of </body>

Body

Paragraphs

The start tag for a paragraph is <p>, and the close tag for the paragraph is </p>.

Paragraphs

Breaks

Within a paragraph, you can use the start tag <br> to skip a line within the text. Keep in mind that <br> does not need a close tag.

Breaks

Lines

Use the <hr> tag to define a thematic change in the content.It needs an end tag.

Lines

Header

The <header> element represents a container for introductory content or a set of navigational links. This requires a close tag, </header>

Headers

Comments

If you want to include ssomething in your code as a reminder, but you don't want it to be visible on your website, you can put a comment in your code by inserting words between the dashes in <!-- -->

Comments

Links

The <a> tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination. For these links, an end tag is not needed, just a close bracket, >.

Links

Images

The <img> tag defines an image in an HTML page. The <img> tag has two required attributes: src and alt.
Example: <img src="smiley.gif" alt="Smiley face" height="42" width="42">

Images

Lists

An unordered list starts with the <ul> tag. Each list item starts with the <li> tag. An ordered list starts with the <ol> tag.

Lists

Tables

The <table> tag defines an HTML tableAn HTML table consists of the <table> element and one or more <tr>, <th>, and <td> elements. The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell.

Tables

Blocks

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can). The <div> element is a block-level element.
Example:
<div>Hello</div>
<div>World</div>

Blocks
